## 2. 핵심 게임플레이 메커니즘

### 2.1 기본 조작
| 요소 | 설명 |
|------|------|
| **공격** | 자동 공격 (플레이어 조작 불필요) |
| **이동** | 가상 조이스틱으로 캐릭터 이동 |
| **조작 방식** | 한 손 조작 가능 |

### 2.2 로그라이크 시스템

#### 스테이지 시작
- 매 판 레벨 1에서 시작
- 이전 진행도의 장비/능력치는 유지

#### 레벨업 시스템
- 경험치 획득 → 레벨업
- 레벨업 시 **3개 스킬 중 1개 선택**
- 스킬 최대 **6개** 장착 가능
- 획득한 스킬은 해당 판에서 버릴 수 없음

#### 스킬 승급
- 각 스킬 최대 **5단계** 승급 가능
- 돌파 조합 달성 시 **진화 스킬**로 변환

### 2.3 스테이지 구조

```
[스테이지 시작]
     ↓
[제한 시간 동안 몬스터 러쉬 생존]
     ↓
[보스 등장] → 필드 몬스터 전멸
     ↓
[보스 처치]
     ↓
[스테이지 클리어 / 보상 획득]
```

### 2.4 맵 유형
| 맵 타입 | 특징 |
|---------|------|
| 무한 맵 | 사방으로 뚫린 넓은 공간 |
| 세로 맵 | 세로로 좁은 통로형 |
| 사각형 맵 | 작은 밀폐 공간 |

## 3. 캐릭터 시스템 (특공대)

### 3.1 특공대 특징
- 각 캐릭터 **고유 전용 스킬** 보유
- **1성 → 6성**까지 승급 가능
- 승급 시점별 스킬 강화:
  - 1성: 기본 스킬 부여
  - 3성: 스킬 강화 1단계
  - 6성: 스킬 강화 2단계

### 3.2 펫 시스템
| 항목 | 내용 |
|------|------|
| 펫 종류 | 9종 |
| 공격 방식 | 펫별 고유 공격 패턴 |
| 패시브 효과 | 플레이어 능력치 버프 |

#### 펫 예시
| 펫 이름 | 공격 | 패시브 (레전드 기준) |
|---------|------|---------------------|
| 개굴개굴 | 독 발사 (200% 데미지) | 치명타 데미지 +10% |

## 4. 장비 시스템

### 4.1 장비 등급 체계 (6단계)

| 등급 | 색상 | 특징 |
|------|------|------|
| 일반 | 회색 | 기본 등급 |
| 우수 | 초록 | - |
| 레어 | 파랑 | - |
| 엘리트 | 보라 | 등급 스킬 해금 |
| 에픽 | 노랑 | 노란색 옵션 해금 |
| 레전드 | 빨강 | 최고 등급 |

### 4.2 무기 티어

| 티어 | 무기 |
|------|------|
| SSS | 혼돈의 검 |
| SS | 파괴의 힘, 빛을 쫓는 자 |
| S | 쿠나이 등 |

### 4.3 장비 합성 시스템

#### 기본 합성 (보라 등급까지)
```
동일 장비 3개 → 상위 등급 1개
```

#### 에픽 합성
```
엘리트+2 장비 2개 + 재물 엘리트 장비 6개 → 에픽 1개
```

#### 레전드 합성
```
에픽+3 장비 1개 + 동일 에픽 장비 2개 → 레전드 1개
```

### 4.4 신의 주조 (레전드 이후)

| 단계 | 필요 재료 |
|------|-----------|
| 1단계 | 기본과학재료 100개 + 종류별 돌 5개 |
| 2단계 | 기본과학재료 200개 + 종류별 돌 10개 |
| 3단계 | 기본과학재료 300개 + 종류별 돌 20개 |

### 4.5 장비 초기화
- +1, +2 강화 롤백 가능
- 레벨업에 사용한 스크롤/자원 반환

## 5. 스킬 시스템

### 5.1 스킬 분류

| 종류 | 설명 | 역할 |
|------|------|------|
| 액티브 스킬 (무기) | 직접 공격 발동 | 데미지 딜링 |
| 패시브 스킬 (지원품) | 버프 효과 | 능력치 강화 |

### 5.2 주요 액티브 스킬

| 스킬명 | 특징 | 용도 |
|--------|------|------|
| 축구공 | 최고 평균 DPS, 튕김 공격 | 보스전 특화 |
| 화염병 | 광역 도트 데미지 | 잡몹 정리 |
| 수호자 | 회전 공격, 발사체 제거, 넉백 | 방어/공격 병행 |
| 쿠나이 | 높은 단일 데미지 | 보스전 |

### 5.3 돌파 조합 시스템

```
[무기 스킬 5성] + [지정 지원품] → [진화 스킬]

예시:
수호자 5성 + 외골격 갑옷 → 수비수 (진화)
축구공 5성 + 운동화 → 양자 공 (진화)
```

### 5.4 진화 조건
- 일반 진화: 공격 스킬 5성 + 보조 스킬 1성 (총 6포인트)
- 특수 진화: 총 10포인트 필요 (더 많은 투자 요구)

## 8. 수익화 모델

### 8.1 하이브리드 수익화 전략

```
[광고 시청] ←→ [인앱결제]
     ↓              ↓
[무과금 플레이]  [프리미엄 경험]
```

### 8.2 주요 과금 상품

| 상품 유형 | 설명 | 특징 |
|-----------|------|------|
| 광고 제거 | 일시불 구매 | 광고 없는 플레이 |
| 챕터 패키지 | 챕터별 한정 상품 | 1회 구매 제한, 높은 가성비 |
| 월정액 카드 | 구독형 상품 | 매일 보상 지급 |
| 일일 패키지 | 데일리 상품 | 매일 구매 가능 |
| 시즌 패스 | 배틀패스 | 무료/유료 트랙 분리 |

### 8.3 상점 구조
- **일일 상점**: 스크롤, 부품, 무기, 장비 판매
- **무료 보석**: 30개/일 + 광고 시청 시 추가 30개

### 8.4 과금 특징
- **무과금 친화적**: 과금 없이도 모든 콘텐츠 플레이 가능
- **광고 기반 수익**: 광고 시청으로 추가 보상 획득
- **박리다매 전략**: 소액 과금 유도

### 8.5 수익 성과

| 지표 | 수치 |
|------|------|
| 글로벌 누적 인앱결제 | 5억 달러+ |
| Habby 전체 수익 중 비중 | 45%+ |
| 한국 시장 기여도 | 약 21% (중국 다음 2위) |
| 한국 시장 수익 | 약 1억 1700만 달러 |

## 10. 핵심 성공 요인 분석

### 10.1 게임성
- **낮은 진입 장벽**: 한 손 조작, 자동 공격
- **깊은 성장 시스템**: 다층적 강화/진화 구조
- **로그라이크 리플레이성**: 매 판 다른 빌드 경험

### 10.2 수익화
- **하이브리드 모델**: 광고 + IAP 병행으로 수익 극대화
- **무과금 친화적**: 유저 풀 확대 후 전환율 확보
- **가성비 상품**: 챕터 패키지 등 소액 상품으로 첫 과금 유도

### 10.3 운영
- **지속적 콘텐츠 업데이트**: 290+ 챕터
- **시즌제 콘텐츠**: 탈출 작전 등으로 리텐션 유지
- **길드 콘텐츠**: 소셜 요소로 이탈 방지
