## 1. 무기 시스템

### 1.1 개요

무기는 플레이어의 기본 공격 수단이며, 각 무기에는 고유한 **기본 스킬**이 내장되어 있습니다.
무기를 장착하면 해당 무기의 기본 스킬이 자동으로 활성화됩니다.

### 1.2 무기 구조

```
┌─────────────────────────────────────────────────────────┐
│                      무기 (Weapon)                       │
├─────────────────────────────────────────────────────────┤
│  • ID: 고유 식별자                                       │
│  • 이름: 무기 표시 이름                                   │
│  • 설명: 무기 설명 텍스트                                 │
│  • 등급: common / rare / epic / legendary               │
│  • 기본 스킬 ID: 무기에 내장된 스킬 참조                   │
└─────────────────────────────────────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────┐
│                      스킬 (Skill)                        │
│              무기 장착 시 자동 활성화                      │
└─────────────────────────────────────────────────────────┘
```

### 1.3 무기 목록

| ID | 이름 | 등급 | 기본 스킬 | 설명 |
|----|------|------|-----------|------|
| weapon_starter_wand | 초보자의 지팡이 | Common | 에너지 볼트 | 기본 지팡이, 에너지 볼트 발사 |
| weapon_spinning_sword | 회전 검 | Common | 회전 검 | 주변을 회전하며 적을 베는 검 |
| weapon_fire_staff | 화염 지팡이 | Rare | 화염 폭발 | 주변에 화염 폭발을 일으킴 |
| weapon_poison_bow | 독 활 | Rare | 독 화살 | 관통하는 독 화살 발사 |
| weapon_lightning_staff | 번개 지팡이 | Epic | 번개 연쇄 | 적 사이를 튕기는 번개 발사 |

### 1.4 무기 장착 흐름

```
[게임 시작]
    │
    ▼
[기본 무기 장착: 초보자의 지팡이]
    │
    ▼
[WeaponSystem.EquipWeapon() 호출]
    │
    ├──▶ [SkillSystem에 기본 스킬 추가]
    │         └─▶ 에너지 볼트 스킬 활성화
    │
    └──▶ [LevelSystem 습득 목록에 추가]
              └─▶ 일시정지 화면에서 확인 가능
```

## 2. 스킬 시스템

### 2.1 개요

스킬은 플레이어의 전투 능력을 결정하는 핵심 요소입니다.
레벨업 시 스킬을 습득하거나 강화할 수 있습니다.

### 2.2 스킬 카테고리

| 카테고리 | 설명 | 예시 |
|----------|------|------|
| **Projectile** | 발사체를 발사하는 스킬 | 에너지 볼트, 독 화살 |
| **Area** | 범위 공격 스킬 | 화염 폭발 |
| **Orbit** | 플레이어 주변을 회전하는 스킬 | 회전 검 |
| **Passive** | 영구 능력치 증가 | 신속 부츠, 강인한 심장 |

### 2.3 스킬 등급

| 등급 | 색상 | 출현 확률 | 설명 |
|------|------|-----------|------|
| Common | 회색 | 높음 | 기본 스킬 |
| Rare | 파란색 | 중간 | 강화된 스킬 |
| Epic | 보라색 | 낮음 | 희귀 스킬 |
| Legendary | 주황색 | 매우 낮음 | 최상위 스킬 |

### 2.4 스킬 목록

#### 2.4.1 공격 스킬

| ID | 이름 | 카테고리 | 등급 | 최대 레벨 | 설명 |
|----|------|----------|------|-----------|------|
| skill_energy_bolt | 에너지 볼트 | Projectile | Common | 5 | 가장 가까운 적에게 에너지 탄환 발사 |
| skill_spinning_blade | 회전 검 | Orbit | Common | 5 | 플레이어 주변을 회전하는 검 |
| skill_fire_burst | 화염 폭발 | Area | Rare | 5 | 주변 범위에 화염 피해 |
| skill_poison_arrow | 독 화살 | Projectile | Rare | 5 | 적을 관통하며 독 피해를 입히는 화살 |
| skill_chain_lightning | 번개 연쇄 | Projectile | Epic | 5 | 적 사이를 튕기는 번개 |

#### 2.4.2 패시브 스킬

| ID | 이름 | 등급 | 최대 레벨 | 효과 |
|----|------|------|-----------|------|
| skill_swift_boots | 신속 부츠 | Common | 5 | 레벨당 이동 속도 5% 증가 |
| skill_vital_heart | 강인한 심장 | Common | 5 | 레벨당 최대 체력 10% 증가 |
| skill_power_gauntlet | 힘의 건틀릿 | Rare | 5 | 레벨당 공격력 8% 증가 |

### 2.5 스킬 레벨업 효과

레벨업 시 스킬의 성능이 향상됩니다:

- **발사체 스킬**: 데미지 증가, 발사체 수 증가
- **범위 스킬**: 데미지 증가, 범위 확대
- **회전 스킬**: 데미지 증가, 회전체 수 증가
- **패시브 스킬**: 효과 수치 증가

## 3. 레벨업 시스템

### 3.1 개요

경험치를 획득하여 레벨업하면 스킬 선택 화면이 표시됩니다.
3가지 스킬 중 하나를 선택하여 습득하거나 강화할 수 있습니다.

### 3.2 경험치 계산

```
필요 경험치 = 기본값 + (레벨 - 1) × 증가량
           = 10 + (레벨 - 1) × 5
```

| 레벨 | 필요 경험치 | 누적 경험치 |
|------|-------------|-------------|
| 1 → 2 | 10 | 10 |
| 2 → 3 | 15 | 25 |
| 3 → 4 | 20 | 45 |
| 4 → 5 | 25 | 70 |
| ... | ... | ... |

### 3.3 스킬 선택 UI 흐름

```
[레벨업 달성]
    │
    ▼
[게임 일시정지]
    │
    ▼
[스킬 선택 화면 표시]
    │
    ├─── 스킬 3개 카드 표시 (상단)
    │         └─ 아이콘, 이름, 신규/레벨 뱃지
    │
    ├─── 선택된 스킬 상세 설명 (중단)
    │         └─ 이름, 등급, 설명, 레벨 변화
    │
    └─── 획득 버튼 (하단)
              │
              ▼
         [스킬 습득/강화]
              │
              ▼
         [게임 재개]
```

### 3.4 스킬 선택 규칙

1. **최대 레벨 미만인 스킬만 표시**: 이미 최대 레벨인 스킬은 선택지에서 제외
2. **랜덤 3개 선택**: 사용 가능한 스킬 중 무작위로 3개 표시
3. **신규 vs 강화 구분**: 처음 습득하는 스킬은 "신규" 표시, 이미 보유 중인 스킬은 레벨 표시

## 5. 캐릭터 시스템

### 5.1 개요

캐릭터는 플레이어가 조작하는 영웅입니다. 각 캐릭터마다 고유한 **기본 무기**와 **스탯**이 다릅니다.
게임 시작 전 캐릭터 선택 화면에서 원하는 캐릭터를 선택할 수 있습니다.

### 5.2 캐릭터 구조

```
┌─────────────────────────────────────────────────────────┐
│                    캐릭터 (Character)                    │
├─────────────────────────────────────────────────────────┤
│  • ID: 고유 식별자                                       │
│  • 이름: 캐릭터 표시 이름                                 │
│  • 설명: 캐릭터 설명 텍스트                               │
│  • 등급: common / rare / epic / legendary               │
│  • 기본 무기 ID: 캐릭터 고유 무기                         │
│  • 기본 스탯: HP, ATK, DEF, SPD, CRIT, C.DMG            │
│  • 대표 색상: 캐릭터 UI 색상                              │
└─────────────────────────────────────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────┐
│                      무기 (Weapon)                       │
│              캐릭터별 고유 무기 자동 장착                   │
└─────────────────────────────────────────────────────────┘
```

### 5.3 캐릭터 등급

| 등급 | 색상 | 설명 |
|------|------|------|
| Common (일반) | 회색 | 기본 캐릭터 |

## 6. 장비 시스템

### 6.1 개요

장비는 플레이어의 기본 스탯을 강화하는 아이템입니다. 각 장비는 **슬롯**에 장착되며, 슬롯당 하나의 장비만 착용할 수 있습니다.
장비는 **강화**를 통해 더 강력하게 만들 수 있습니다.

### 6.2 장비 구조

```
┌─────────────────────────────────────────────────────────┐
│                      장비 (Equipment)                    │
├─────────────────────────────────────────────────────────┤
│  • ID: 고유 식별자                                       │
│  • 이름: 장비 표시 이름                                   │
│  • 설명: 장비 설명 텍스트                                 │
│  • 슬롯: weapon / armor / accessory                     │
│  • 등급: common ~ legendary                             │
│  • 보너스 스탯: HP, ATK, DEF, SPD, CRIT, C.DMG          │
└─────────────────────────────────────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────┐
│                    장비 인스턴스                          │
│              (인벤토리 내 개별 장비)                       │
├─────────────────────────────────────────────────────────┤
│  • 강화 레벨: 1 ~ 최대 레벨                               │
│  • 장착 상태: 장착됨 / 미장착                             │
└─────────────────────────────────────────────────────────┘
```

### 6.3 장비 슬롯

| 슬롯 | 아이콘 | 설명 |
|------|--------|------|

### 6.6 장비 강화

강화 시 장비의 보너스 스탯이 증가합니다.

```
강화 스탯 = 기본 스탯 × (1 + (레벨 - 1) × 0.1)
```

| 강화 레벨 | 스탯 배율 |
|-----------|----------|
| Lv.1 | 100% |
| Lv.5 | 140% |
| Lv.10 | 190% |
| Lv.15 | 240% |
| Lv.20 | 290% |
| Lv.25 | 340% |
| Lv.30 | 390% |
